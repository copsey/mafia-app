Cheatsheet
----------

Preprocessing:
  {example}      -- substitute a parameter named 'example'
  {!if p}        -- begin "if" block for boolean "p"
  {!else_if q}   -- continue "if" block with another condition on boolean "q"
  {!else}        -- continue "if" block with default path
  {!list stuff}  -- repeat the text in this block for all members of "stuff"
  {!end}         -- end previous block
  {-Note 1}      -- comment, stripped from text
  {--- Note 2}   -- ... anything that begins with '{-' is a comment

Text formatting:
  =Main Menu=    -- title
  _really_       -- italics
  %How to play%  -- help text
  %Help message  -- ... it doesn't need to end with a '%' sign
  @ok@           -- monospace typeface

Escape sequences:
  \\  -- backslash
  \{  -- open curly brace
  \}  -- close curly brace
  \=  -- equals sign
  \_  -- underscore
  \%  -- percent sign
  \@  -- at sign
