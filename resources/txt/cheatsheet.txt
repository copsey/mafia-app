 Cheatsheet
============

Preprocessing:
  {example}      -- substitute a parameter named 'example'
                 -- ... it must be either a string or a number
  {!if p}        -- begin "if" block
  {!if x < 5}    -- use a comparison instead of a boolean
  {!if 5 > x}    -- ... each side can be either a parameter or a constant
  {!if x = y}    -- ... it's also possible to check for equality
  {!else_if q}   -- continue "if" block with another condition
  {!else}        -- continue "if" block with default path
  {!list stuff}  -- repeat the text in this block for all members of "stuff"
  {!end}         -- end previous block
  {-Note 1}      -- comment, stripped from text
  {--- Note 2}   -- ... anything that begins with '{-' is a comment

Text formatting:
  _Proper noun_  -- italics
  *important*    -- bold
  @ok@           -- monospace typeface
  =Main Menu=    -- title
  %How to play%  -- help text
  %Help message  -- ... it doesn't need to end with a '%' sign
  ~fancy~        -- flavour text

Escape sequences:
  \\  -- backslash
  \_  -- underscore
  \*  -- asterisk
  \@  -- at sign
  \=  -- equals sign
  \%  -- percent sign
  \~  -- tilde
  \{  -- left curly brace
  \}  -- right curly brace
